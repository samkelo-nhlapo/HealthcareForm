# Legacy Migration Package (Archived)

This folder contains the archived documentation for the historical migration from ASP.NET MVC 5 (.NET Framework 4.7.2) to ASP.NET Core 6.0.

Why it is archived:
- The current backend targets `net10.0` and already runs on modern .NET.
- Keeping these files in the main backend root created confusion with current build/deploy docs.

Use this folder only if you are auditing or recreating the older migration work.

Primary entry points:
- `INDEX_MIGRATION.md`
- `QUICK_REFERENCE.md`
- `MIGRATION_GUIDE.md`
